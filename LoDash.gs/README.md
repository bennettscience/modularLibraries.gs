# Lodash.gs

Use the very useful and popular library in your Google Apps Scripts

## Quickstart

Copy the [code]() into your project. Use it:

```js
var _ = Import.LoDash();
```

Or set it globally:

```js
Import.LoDash({
   namespace: ""
});
```

Consult the [documentation](https://lodash.com/docs/4.17.11) for methods.
